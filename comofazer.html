Como fazer uma DIV

Usa para agrupar outros elementos juntos, tipo uma “caixinha” que organiza a página.

Como botar uma IMAGEM

Mostra uma imagem na página. Precisa do atributo src com o endereço da imagem.

"src" é o nome do arquivo, "alt" é o nome que voce vai chama ela no codigo

Pra que serve a SECTION

Serve pra separar uma parte do conteúdo, tipo um capítulo ou área específica da página.

Pra que serve o PARAGRAFO

É um parágrafo de texto

Pra que serve o BR

Serve para pular a linha e


ir para proxima

Como fazer um botão

Cria um link pra outra página ou site. Precisa do atributo href

Oque é o BODY

É onde fica todo o conteúdo visível da página (textos, imagens, botões).É onde fica todo o conteúdo visível da página como textos, imagens, botões

Oque é o HEAD

Como usar o MAIN

Marca o conteúdo principal da página, diferente de cabeçalho, rodapé ou menu.
