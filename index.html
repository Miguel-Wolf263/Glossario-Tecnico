HTML Tags

Como fazer

É uma tag genérica usada para agrupar elementos e aplicar estilos ou scripts.

<div>Conteúdo agrupado</div>

Tag para definir e exibir uma imagem na página.

<img src="imagem.jpg" alt="Descrição da imagem">

Usada para dividir o conteúdo em seções com significado, como tópicos ou capítulos.

<section>Tags html</section>

Cria parágrafos de texto.

<p>Um parágrafo de texto.</p>

Quebra a linha e pula para a próxima.

<br>

Cria links clicáveis.

<a href="https://exemplo.com">Clique aqui</a>

Contém todo o conteúdo visível do site.

<body>Conteúdo visível da página "head/header"</body>

Cria títulos e subtítulos (H1 é o mais importante).

<h1>Título Principal</h1>

Guarda informações invisíveis da página, como título e links de estilo.

<head><title>Glossario tecnico</title></head>

Marca o conteúdo principal da página, que não se repete como menus ou rodapés.

<main>Conteúdo principal,quase todo o codigo</main>

CSS Propriedades

Como fazer

Muda a cor de um texto ou elemento.

p { color: red; }

Define o espaço externo entre o elemento e os outros.

p { margin: 20px; }

Controla como o elemento aparece na página (inline, block, flex, etc).

div { display: flex; }

Muda a cor do fundo da pagina.

div { background-color: yellow; }

Controla o tamanho da fonte do texto.

h1 { font-size: 24px; }

Controla o espaço DENTRO do elemento.

div { padding: 10px; }

Poem uma borda ENVOLTA do elemento.

div { border: 1px solid black; }

Alinha o texto dependendo da direção que voce escolher(Left,Rigth,Center etc...).

h2 { text-align: center; }

Define a largura de um elemento.

div { width: 300px; }

Define a ALTURA de um elemento.

div { height: 150px; }
